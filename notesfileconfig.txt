"root": Especifica la ruta del sistema de archivos donde se encuentran los archivos del sitio web.
"listen": Define en qué dirección IP y puerto debe escuchar el servidor web las solicitudes entrantes.
"server_name": Especifica el nombre del servidor virtual o dominio al que responde este bloque de configuración.
"error_page": Define la página que se debe mostrar en caso de que ocurra un error específico.
"client_max_body_size": Limita el tamaño máximo de la solicitud que el servidor web está dispuesto a procesar.
"upload_store": Define la ubicación en el sistema de archivos donde se guardarán los archivos cargados por los usuarios.
"index": Especifica el nombre de los archivos que el servidor web debe buscar automáticamente cuando se accede a un directorio.
"autoindex": Habilita o deshabilita la generación automática de listados de directorios.
"return": Permite al servidor web enviar respuestas de redireccionamiento o respuestas personalizadas.
"allow_methods": Especifica los métodos de solicitud HTTP permitidos para una ubicación particular.
"location": Define una ubicación dentro del servidor web y establece la configuración específica para esa ubicación.
